# Diversidade: seja você, seja feliz!

O aplicativo Diversidade é fruto da disciplina Projeto Social do Instituto Federal de Educação, Ciência e Tecnologia do Ceará. A proposta do aplicativo é mapear locais e eventos que ofereçam serviços gratuitos de apoio à comunidade LGBT no Ceará. Esses serviços podem se encaixar nas categorias principais: saúde, educação, direitos, lazer, cultura, socialização e convívio. Esperamos que essa iniciativa contribua para a inserção das pessoas LGBTs em sua comunidade, para que assim todo o grupo se fortaleça.


## Instruções
- Desenvolvido no Xcode 7.3

Após clonar o repositório, você deve instalar as dependencias do cocoa pods
> pod install


Depois, basta instalar as dependencias do projeto e executá-lo.

## Funcionalidades
Colabore com novas funcionalidades e venha aprimorar essa ferramenta de Cidadania.

- [x] Visualizar locais ou eventos de apoio à comunidade LGBTT do seu estado
- [x] Vizualizar informações básicas sobre um ponto no mapa
- [x] Obter localização e contatos de um determinado local/evento
- [x] Visualizar os serviços ofertados por um determinado local/evento
- [x] Visualizar e realizar comentários sobre um determinado local/evento
- [x] Obter informações sobre Direitos LGBTT
- [x] Obter informações sobre Saúde LGBTT
- [x] Obter informações sobre Descobrindo-se LGBTT
